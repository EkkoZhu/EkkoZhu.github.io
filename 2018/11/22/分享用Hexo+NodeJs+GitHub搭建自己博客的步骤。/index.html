分享用Hexo+NodeJs+GitHub搭建自己博客的步骤。

2018-11-22  

在安装Hexo之前，你必须安装这两个应用：

\1. Node.Js 链接：http://nodejs.cn/download/ 傻瓜式安装。

这一步选第四个表示自动配置环境变量。（这个截图是网上找的因为我自己安装时没有截图）

\2. Git 链接https://git-scm.com/download/win 进入这个页面会弹出一个下载框。安装也是一路点Next。

Win+R 输出cmd进入命令行 ，成功界面如下：

现在开始安装Hexo：

\1. 首先在自己电脑上创建一个文件夹用于存放Hexo。

\2. 打开git-base快捷方式。也可以在桌面鼠标右键选git-base（如果没有则安装有问题）

\1. 输入cd 文件下路径

\2. 输出npm install -g hexo-cli 一键安装Hexo

\3. 输入hexo -v，检查hexo是否安装成功

\4. 输出 hexo init 初始化 （等待一会儿会看到 INFO “Start blogging with Hexo！” 完成！ 此时可以在刚才文件夹中看到生成的文件）

\5. 输入npm install，安装所需要的组件

\6. 输入hexo g，首次体验Hexo

\7. 输入hexo s，开启服务器，访问该网址（一般是localhost：4000），正式体验Hexo。出现下图就是成功了

最后一步 ，把博客放到GitHub上。

\1. 访问 https://github.com/ 创建账号 登录。创建存储库：

这里要注意，命名一定要是这个格式，这将是你之后的博客访问地址。因为我已经创建过这个了所以这里显示错误。

\2. 回到gitbash中，

\3. 输入npm install hexo-deployer-git –save 这样才能将你的文章发布到GitHub上。

\4. 配置github账户信息（YourName和YourEail都替换成你自己的）

\5. 输入 ssh-keygen -t rsa -C “youremail@example.com 生成SSH

\6. 通过如下指令获取 id_rsa.pub文件的内容，也就是ssh密匙。（一般这个文件生成在C盘用户组，也可以手动打开文件复制）

\7. 将复制的文件放在这里

\8. 输入ssh -T git@github.com 测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了。

\9. 执行以下命令（建议每次都按照如下步骤进行部署）

hexo clean

hexo generate

hexo deploy

可简写为

hexo c

hexo g

hexo d

最后看到 INFO Deploy done: git

就ok了。

建议：在使用命令 hexo g 部署之后，可以先使用 hexo s 运行本地站点，在浏览器输入地址http://lacolhost:4000/查看运行结果，检查没有错误后再使用命令 hexo d 发布

此时打开浏览器http://你的用户名.github.io就可以访问你的博客了。

比如我的是：<https://ekkozhu.github.io/>
