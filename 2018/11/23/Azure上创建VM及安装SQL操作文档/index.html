Azure上创建VM及安装SQL操作文档

2018-11-23   azure 

说明：

本篇操作文档主要分为两个步骤展现：

\1. 在Azure上创建虚拟机。

\2. 安装SQL Server数据库。

下面会详细分解每一个步骤的具体操作。

一、 在Azure上创建虚拟机。

\1. 登录到 Azure 门户 http://portal.azure.cn

\2. 先点击“资源组”，根据个人需求新建资源组或选择已有的资源组。©2018 CloudRiches All rights reserved.

\3. 选择资源组后点击“创建资源”，选择虚拟机Windows Server——右边再选择“Windows Server 2012 Datacenter”_最后确定创建即可。

\4. 输入虚拟机名称，区域，可用性选项选为“可用性集”。可用性集下面点击“新建”，输入一个名称后点击确定。

然后点击“更改大小”配置虚拟机内存大小，搜索ds14点击DS14 v2 后点击下面的选择即可。

\5. 输入这个虚拟机的用户名和密码，下面的几个选项如下图，只有点击下一步：磁盘

\6. 选择磁盘类型“标准SSD”，点击“创建并附加新磁盘”，然后这个类型也是选择“标准SSD”，点击确定。点击下一步。

\7. 下一步-管理-来宾配置-标记 都是默认配置直接点下一步。最后在“查看和创建”页面核对信息，确定无误后点击“创建”。稍等一会即可看到“部署完成”。

二、 安装数据库

1.点击资源组，选择之前创建的资源组，可以看到刚才新建的虚拟机，点进去后点击 连接。

2.如图，点击“下载RDP文件”，打开，弹出远程连接窗口。选择连接

\3. 连接上了之后，点击左下角，选择浏览器，打开

https://download.my.visualstudio.com/sg/ct_sql_server_2012_standard_edition_x86_x64_dvd_813406.iso?t=7e030917-4ced-47b7-a5a9-9fa49be2c80b&e=1542720757&h=061f8dc342f1d11f16fc978e9ce0876d&su=1

这个链接下载SQL Server 2012。

\4. 下载完成后在文件中找到setup双击打开。

选择你想要安装的，一般都是安装全新SQL Server独立安装。（第一个）

点击之后等电脑反应一会儿就好了。这里会自动生成密匙，然后直接下一步。

下一步 等待..

这里全选。然后下一步。

继续点下一步下一步。。

这里选择混合模式进行登录，并添加当前用户。然后点击“下一步”。

一直点下一步，看到这个页面后，点击install安装。

这个安装过程比较慢，请耐心等待。
